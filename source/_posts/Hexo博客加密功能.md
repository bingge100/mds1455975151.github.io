# 前言
如何给博客内容添加访问限制，保证信息安全？

# 部署
## 安装插件
Github地址:https://github.com/MikeCoder/hexo-blog-encrypt
``` bash中使用如下命令进行发布
cd hexo
npm install hexo-blog-encrypt --save
```

## 配置插件
[参考插件官网说明](https://github.com/MikeCoder/hexo-blog-encrypt)

## 查看修改效果
```bash中使用如下命令进行发布
hexo g
hexo s
```
在本地访问 http://127.0.0.1:4000
